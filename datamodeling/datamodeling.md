## 데이터 모델링 이란?
- Model? : 어떤 목적을 가지고 진짜를 모방한것
- **목표** : 컴퓨터(데이터베이스(관계형데이터베이스))에 정보담기  즉 **'데이터베이스 표에 정보를 담는것'** 
- 목적 : 인간이 다루이 어려운 많은양의 데이터를 빠르게 다룰수 있게 됨
- 결국 데이터 모델링이란 복잡한 현실을 컴퓨터에 담는것 = **데이터모델링**

## 개념적 데이터 모델링
- 파악한 업무에서 개념을 뽑아내는 것 이것을 도와주는 것이 ERD이다.
- ERD의 기능
    - 정보를 발견하고 표현
    - 연관된 정보를 인식하고 표현
    - 정보그룹간의 관계인식 및 표현
    - 표로 전환기능
